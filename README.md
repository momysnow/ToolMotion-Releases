# ToolMotion Releases

Official release binaries for [ToolMotion](https://toolmotion.dev) - Professional G-Code Editor.

## Downloads

Visit the [Releases](https://github.com/momysnow/ToolMotion-Releases/releases) page to download the latest version.

### Available Platforms
- **macOS** (Apple Silicon) - `.dmg` installer
- **Windows** (coming soon) - `.exe` installer

## Installation

### macOS
1. Download the `.dmg` file
2. Open the downloaded file
3. Drag ToolMotion to your Applications folder
4. Right-click and select "Open" on first launch (required for unsigned apps)

### Windows
1. Download the `.exe` installer
2. Run the installer
3. Follow the installation wizard
